My Favorite HTML Tags

Intro. This webpage is to show some of my favorite tags to use in HTML.

<a>. The anchor tag is great because it allows you to hyperlink websites within your webpages. Like this! Being able to hyperlink will make your website that much more accessible.

<form>. The form tag is one of the most useful and most used tags there is. It is used to get information from the user accessing the page. An example of this can be seen below with the "Course lookup".

<style>. Many beginer web developers often forget to use the style tag. This tag helps to make your webpages more appealing to the eye. For instance, I used the style tag on this page to make the background light grey, give the main body two columns, and to create a max width of 750 pixels. It can be used to style any specific area within your page.

Conclusion. These three tags are incredibly useful when creating webpages. They are all used on almost every single webpage you visit. From the simplest site to the most complex, these tags are used to help make any page more accessible.
