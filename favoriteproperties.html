Introduction.

This webpage shows 5 different CSS properties and their use in websites.

The "border" Property

The "border" property allows us to create borders and use all of the border properties in one place.

The "animation" Property

The "animation" property puts a few other CSS properties into a single one. It uses timing-function, delay, iteration-count, direction, fill-mode, name, duration, and play-state. The following square uses the pulse animation.

The "font" Property

The "font" property again puts multiple properties into one single property. It uses style, variant, weight, size, and family.This helps to give distinctive looks to certian portions of your webpage.

The "text-align" Property

The "text-align" property is extremely important when organizing your page correctly and putting it in a format that is familiar to the user. An example follows this paragraph.
