Container

I used the container class to organize my website in a logic fashion. This class is essential when creating a appealing website.

Container-fluid

This class works directly with the container class. I used it because it is a container that takes up the full width of the webpage. This is very useful when designing a website compatable with mobile devices.

Carousel-slide

The Carousel-slide class is widely used across the internet and for good reason. I personally used it because it is a great way to show the user what the website is about in a visually pleasing way.

Carousel-caption

Working with the carousel-slide class, the carousel-caption class is a quick way to add relavent text to your slider pictures.

Collapse

I used the collapse class as a way to condense the navbar when viewing in a small window or mobile device. This makes for much easier navigation, no matter what device you are viewing the website from.

Navbar

My choice to use the navbar class was because of its obvious usefulness. It is a simple and effective way to navigate throughout your website.

Navbar-brand

I used the navbar brand to show the user the logo of the website without being too obnoxious with it.

Border

The border class was used by me to make sure that the navbar fit exactly how expected on the site.

col-sm

Used to define the size of the screen for tablets. I used it to help make the website more compatible with most, if not all, devices.

col-lg

I used it to define the screen size for laptops and desktops. I used it again to help make the website more compatible with most, if not all, devices.

Pros/Cons

For anyone not fluent in coding, making a website with bootstrap would be a nightmare, despite its many qualities that are attractive to programmers.Wordpress on the other hand is great for the casual website developer. It's simple app-like approach is easy for anyone to understand and modify their site with. Where with bootstrap you need to dive into code to edit your site. Also wordpress uses a built in login system that allows you to sete up blogs and comment sections without the need to personally connect a database.

If anyone is jumping on the wordpress train after reading that paragraph, they need to look more into the debate. Bootstrap also has its massive benifits. For one, it may be confusing on how to build a wordpress at first, but once you understand it there are limitless possibilites for your website. Wordpress limits the look of your website by forcing you to use premade templates and bootstrap lets you start from scratch and your options for creativity are endless.
